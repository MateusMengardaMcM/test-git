
# Resumos Git e Github

Projeto voltado para fins educacionais, com intenção de aprender e compreender como utilizar o Git e o Github.


## Aprendizados

Compreendi como criar e modificar arquivos README pro meio do Gitbash, além de compreender o comando **git.status**.
## Funcionalidades

- Apresenta uma lista para os comados Git mais comuns

## Stack utilizada

**Front-end:** Html



## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone URL
```
